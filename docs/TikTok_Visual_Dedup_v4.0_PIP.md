## 一、方案概述

### 1.1 核心思路

将 **背景模糊 + 画中画** 组合变成一个**必须实施的模式**，通过三层视频架构实现：
- **最大化去重效果**（背景层采用强化版处理）
- **保持观感质量**（中间层保持清晰）
- **二创高级感**（文字层显示字幕）

### 1.2 核心决策（已确认）

| 问题 | 决策 |
|------|------|
| 背景变速 vs 中间层 | **同步变速** |
| 文字层内容 | **字幕提取**（无字幕则不显示） |
| 音频处理 | **保持原声** |
| 与 v3.0 关系 | **v4.0 替代 v3.0** |

## 二、三层架构详解

### 2.1 背景层（去重核心）

**作用**：承担主要去重任务

#### 2.1.1 必做特效（7项全做）

| 序号 | 特效名称 | 参数范围 | 说明 |
|------|----------|----------|------|
| 1 | **全景模糊** | σ = 40-60 | 高斯模糊，不可识别背景内容 |
| 2 | **暗化** | -0.3 ~ -0.1 | 降低背景亮度 |
| 3 | **放大** | 130-180% | 确保填充整个画面 |
| 4 | **变速** | 1.05-1.2x | 整体同步变速（背景+中间层+音频） |
| 5 | **镜像翻转** | 50% 概率 | 水平翻转 |
| 6 | **随机裁剪** | 3-5%（上下左右随机） | 裁剪边缘 |
| 7 | **随机旋转** | ±10° | 轻微旋转 |

#### 2.1.2 增强特效（从7项中选3-5个）

| 序号 | 特效名称 | 参数范围 | 说明 |
|------|----------|----------|------|
| 1 | 饱和度 | 0.90-1.15 | eq=saturation={val} |
| 2 | 亮度 | -0.10-0.15 | eq=brightness={val} |
| 3 | 对比度 | 0.90-1.15 | eq=contrast={val} |
| 4 | RGB 偏移 | 2-5 | colorchannelmixer |
| 5 | 色调统一 | 色温 ±10% | 改变色彩指纹 |
| 6 | 帧交换 | 随机相邻帧 | 交换相邻帧位置 |

### 2.2 中间层（观感保护区）

**作用**：保持原视频清晰度，与背景同步变速

| 处理项目 | 参数范围 | 目的 |
|----------|----------|------|
| **缩放** | 与背景层同比例（130-180%） | 保持视觉一致性 |
| **暗化** | -0.1 ~ -0.05 | 轻微暗化，增强层次感 |
| **裁剪** | 上下裁剪，剩余 60-70% | 去除上下边缘 |
| **边框** | 50% 概率，14-18px，只有上下边缘 | 增加二创感 |

**边框颜色**：白/黄/浅蓝/浅紫（随机选择）

### 2.3 文字层（字幕显示）

**作用**：显示提取的字幕，增加二创感

**触发条件**：仅当视频有字幕时显示此层，无字幕则跳过

| 元素 | 设计规范 |
|------|----------|
| **内容来源** | WhisperX 字幕提取 |
| **位置** | 底部（距底部 80-120px） |
| **配色** | 深灰底（#1a1a1a@0.8）+ 白字（#ffffff） |
| **字体** | 加粗，字号 36-48 |

## 三、技术实现

### 3.1 FFmpeg 命令模板

```bash
ffmpeg -i input.mp4 \
  -filter_complex "
    # 背景层：放大 + 全景模糊 + 暗化 + 必做效果
    [0:v]scale=iw*1.5:ih*1.5,
         gblur=sigma=50,
         eq=brightness=-0.2,
         rotate=0.05:c=black,
         crop=iw*0.94:ih*0.94,
         setpts=0.90*PTS[bg];
    
    # 中间层：同比例放大 + 暗化 + 裁剪 + 可选边框
    [0:v]scale=iw*1.5:ih*1.5,
         eq=brightness=-0.08,
         crop=iw:ih*0.65:0:ih*0.18,
         pad=iw:ih+32:0:16:color=white,
         setpts=0.90*PTS[fg];
    
    # 合成：背景层 + 中间层
    [bg][fg]overlay=(W-w)/2:(H-h)/2[video];
    
    # 帧率调整（简化抽帧）
    [video]fps=fps=29.97[v_dropped]
  " \
  -map "[v_dropped]" -map 0:a \
  -filter:a "atempo=1.11" \
  -c:v mpeg4 -q:v 8 \
  -c:a aac \
  -y output_pip.mp4
```

## 四、去重效果分析

### 4.1 技术层面

| 检测维度 | 改变方式 | 效果 |
|----------|----------|------|
| **MD5 哈希** | 三层合成 + 重新编码 | ✅ 100% 不同 |
| **帧级匹配** | 背景像素 100% 改变 + 中间层偏移 + 抽帧 | ✅ 极高差异 |
| **色彩指纹** | 整体色调被背景层主导 + 暗化处理 | ✅ 完全不同 |
| **时域特征** | 同步变速 1.05-1.2x + 抽帧 | ✅ 时域差异 |
| **感知哈希 (pHash)** | 画面结构重组 + 旋转 + 裁剪 | ✅ 显著差异 |

### 4.2 感知层面

| 心理机制 | 应用方式 | 效果 |
|----------|----------|------|
| **视觉注意力** | 人眼聚焦中心区域 | 只看到清晰区域 |
| **边缘视觉** | 边缘视觉对模糊不敏感 | 背景处理不被注意 |
| **完形心理** | 大脑自动补全背景信息 | 模糊背景被视为自然 |
