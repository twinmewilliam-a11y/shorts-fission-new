# 文字层 v2.1 设计方案 - 更有设计感的模板

## 当前问题
- 当前特效主要是字体颜色、描边、阴影
- 缺少**框体设计、背景块、装饰元素**
- 整体设计感不够

## 新增模板设计（3种）

---

### 🎯 模板 A: 运动标签框（Sports Label）

**适用场景**: 体育赛事、运动集锦

**设计元素**:
```
┌─────────────────────────────────────┐
│  ▶ 精彩进球！                        │
└─────────────────────────────────────┘
```

**CSS 实现**:
- 左侧箭头装饰 `▶`
- 半透明深色背景块（rgba(0,0,0,0.6)）
- 圆角边框（border-radius: 8px）
- 黄色/白色文字
- 内边距（padding: 8px 16px）

**ASS 实现**:
```
\1a&H60&           ; 背景透明度 60%
\4a&H60&\4c&H000000&  ; 阴影作为背景块
\bord8\shad0       ; 粗边框模拟背景
\c&HFFFF00&        ; 黄色文字
```

---

### 🎯 模板 B: 访谈字幕卡（Interview Card）

**适用场景**: 体育访谈、对话分析

**设计元素**:
```
╔═════════════════════════════════╗
║  "这是一段访谈内容"              ║
║                                  ║
║            —— 受访者             ║
╚═════════════════════════════════╝
```

**CSS 实现**:
- 双线边框（border: 3px double #fff）
- 米黄色背景（rgba(255,248,220,0.8)）
- 引号装饰 `"`
- 署名右对齐

**ASS 实现**:
```
\bord3             ; 粗边框
\3c&HFFFFFF&       ; 白色描边
\c&HF5DEB5&        ; 米黄色文字
" + text + "       ; 引号
```

---

### 🎯 模板 C: 漫剧气泡框（Drama Bubble）

**适用场景**: 短剧、漫剧、情感内容

**设计元素**:
```
    ╭──────────────────────────────╮
    │  这是一段感人的台词          │
    ╰──────────────────────────────╯
           ○
```

**CSS 实现**:
- 圆角矩形背景（border-radius: 16px）
- 白色背景块（rgba(255,255,255,0.9)）
- 底部小三角装饰
- 黑色文字

**ASS 实现**:
```
\1a&H10&           ; 背景几乎不透明
\4c&HFFFFFF&       ; 白色背景
\bord1\shad2       ; 轻微阴影
\c&H000000&        ; 黑色文字
\q2                ; 换行模式
```

---

### 🎯 模板 D: 品牌横幅（Brand Banner）

**适用场景**: 品牌宣传、广告引流

**设计元素**:
```
╔═══════════════════════════════════════╗
║  🔥 限时优惠！点击链接  🔥              ║
╚═══════════════════════════════════════╝
```

**CSS 实现**:
- 全宽背景块
- 渐变背景（linear-gradient）
- 左右 emoji 装饰
- 白色粗体文字

**ASS 实现**:
```
\bord6             ; 粗边框
\3c&HFF6600&       ; 橙色边框
\c&HFFFFFF&        ; 白色文字
\b1                ; 粗体
🔥 + text + 🔥
```

---

### 🎯 模板 E: 渐变发光字（Gradient Glow）

**适用场景**: 潮流内容、音乐视频

**设计元素**:
```
    ✨ 这是渐变发光文字 ✨
    ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
```

**CSS 实现**:
- 文字渐变（background-clip: text）
- 外发光效果（text-shadow: 0 0 20px #fff）
- 星星装饰

**ASS 实现**:
```
\blur3             ; 模糊发光
\3c&H00FFFF&       ; 青色描边
\4c&HFF00FF&       ; 粉色阴影
\c&HFFFFFF&        ; 白色文字
```
