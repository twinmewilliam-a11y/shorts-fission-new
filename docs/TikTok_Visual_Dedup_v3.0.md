### 5.2 完整 FFmpeg 命令

```bash
ffmpeg -i input.mp4 \
  -vf "hflip,rotate=0.2618:c=black,scale=iw*1.035:ih*1.035,crop=iw*0.7:ih*0.7:iw*0.15:ih*0.15,eq=saturation=1.05,eq=contrast=1.08" \
  -filter:a "atempo=1.03" \
  -c:v mpeg4 -q:v 5 \
  -c:a aac \
  -y output.mp4
```
