# shorts-fission - 变更日志

## v4.0 - 2026-03-13

### 🎯 核心升级：背景模糊+画中画（PIP）方案

**架构变更**：v4.0 **替代** v3.0，采用三层合成架构

### 新特性

**三层架构**
- **背景层**：全景模糊 + 强化处理（7项必做 + 增强效果）
- **中间层**：画中画模式（两种可选）
  - `portrait_crop`：竖屏裁剪（默认）
  - `landscape`：横屏显示（新增）
- **文字层**：字幕烧录（ASS override 标签精确位置控制）

**参数调整（v2）**
- 背景层暗化：去除（0）
- 背景层放大：130-160%
- 中间层暗化：去除（0）
- 中间层裁剪：55-65%
- 新增 landscape 模式：横屏显示，屏幕占比 55-65%

### 修复
- 字幕位置精确控制（使用 ASS override 标签）
- 字幕透明度提升（&H50 → &HB0，约70%不透明）
- variant_index 重复问题（使用 actual_index）
- variant_count 累加逻辑
- FFmpeg 背景模糊参数（boxblur → gblur）

### 优化
- FFmpeg 超时：600秒 → 1800秒
- 编码质量：q:v 5 → q:v 8
- 前端三层参数分区显示
- 新增「中间层横屏模式」checkbox

### 集成
- yt-dlp-api 代理服务（端口 8001）
- Scrapling 下载器（绕过 Cloudflare）

## v3.0 - 2026-03-08

### 两步策略变体引擎

**已废弃**：被 v4.0 替代

**步骤1：基础必做（6项）**
1. 水平镜像
2. 随机旋转（±5°）
3. 随机缩放（95%-105%）
4. 变速（0.9x-1.1x）
5. 随机裁剪
6. 掐头去尾

**步骤2：增强组合（从9项中选3-5个）**
1. 饱和度
2. 亮度
3. 对比度
4. RGB 偏移
5. 高斯模糊
6. 抽帧
7. 帧交换
8. 画中画
9. 背景模糊（新增）
